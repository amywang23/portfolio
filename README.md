# Dynamic Portfolio (React + Vite)

This is a dynamic, recruiter-friendly portfolio with animations, particle background, command palette, draggable project cards, filters, and a modal case study.

## Quickstart
```bash
npm install
npm run dev
```

## Deploy to GitHub Pages
1) Edit `vite.config.js` and set `base` to your repo name:
```js
base: "/YOUR_REPO_NAME/",
```

2) Deploy:
```bash
npm run deploy
```
